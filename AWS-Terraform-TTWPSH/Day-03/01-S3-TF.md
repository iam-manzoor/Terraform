# Provision S3 using Terraform

- Provision S3 bucket using Terraform
  - Write the TF file
  - ```
    terraform {
      required_providers {
        aws = {
          source = "hashicorp/aws"
          version = "~> 6.0"
        }
      }
    }

    provider "aws" {
      region = "use-east-1"
    }

    resource "aws_s3_bucket" "demo_bucket" {
      bucket = "my-s3-bucket-1234"
    }

    terraform init
    terraform plan
    terraform apply
    ```
